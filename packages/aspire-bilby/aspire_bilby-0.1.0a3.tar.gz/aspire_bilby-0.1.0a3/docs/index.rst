aspire-bilby: Bilby interface for aspire
=======================================

``aspire-bilby`` lets you run ``aspire`` as a sampler inside ``bilby`` and
convert between ``bilby`` results and ``aspire`` workflows. It reuses the
core ``aspire`` sampling and flow-fitting utilities while integrating with
``bilby`` inputs and outputs.
