# dp_batch_ensemble.py

This master Python script batch-executes a set of DP ensemble simulations for a range of grid sizes, ranging from 31 $\times$ 31 up to 2000 $\times$ 2000; each ensemble varies the linear DP coefficient $a$ for 15 values over a narrow range around criticality.

See: [`Langevin/simulation/dp/dp_batch_ensemble.py`](https://github.com/cstarkjp/Langevin/tree/main/simulation/dp/dp_batch_ensemble.py)
