Statistiky

Co sledujete?

Sledujeme dva typy událostí:

  1. Návštěvy stránky záznamu.
  2. Stažení souboru.

U obou typů událostí sledujeme:

  1. Návštěvník: Anonymizované ID návštěvníka.
  2. Typ návštěvníka: Zda byl požadavek proveden a) člověkem, b) strojem nebo c) robotem.
  3. Země: Země původu požadavku (na základě IP adresy).
  4. Odkaz: Doména odkazu.

Co je zobrazení ?

Uživatel (člověk nebo stroj) navštěvující záznam, bez započítání dvojitých kliknutí a robotů.

Co je unikátní zobrazení ?

Unikátní zobrazení je definováno jako jedna nebo více návštěv uživatele v časovém okně 1 hodiny. To znamená, že pokud byl stejný záznam navštíven vícekrát stejným uživatelem ve stejném časovém okně, považujeme to za jedno unikátní zobrazení.

Co je stažení ?

Uživatel (člověk nebo stroj) stahující soubor ze záznamu, bez započítání dvojitých kliknutí a robotů. Pokud má záznam více souborů a stáhnete všechny soubory, každý soubor se počítá jako jedno stažení.

Co je unikátní stažení ?

Unikátní stažení je definováno jako jedno nebo více stažení souborů z jednoho záznamu uživatelem v časovém okně 1 hodiny. To znamená, že pokud byl jeden nebo více souborů stejného záznamu stažen vícekrát stejným uživatelem ve stejném časovém okně, považujeme to za jedno unikátní stažení.

Co je objem stažených dat ?

Celkový objem dat, který byl stažen pro všechny soubory v záznamu uživatelem (člověkem nebo strojem), bez započítání dvojitých kliknutí a robotů. V případě, že uživatel zruší stahování souboru v průběhu, stále počítáme celkovou velikost souboru jako plně staženou.

Jak nakládáte s verzemi?

Ve výchozím nastavení u záznamu zobrazujeme agregované počty zobrazení, stažení a objemu dat pro všechny verze záznamu. Můžete rozbalit box se statistikami použití na stránce záznamu a zobrazit počty pro konkrétní verzi.

Jak nakládáte s roboty?

Požadavky provedené roboty (také známými jako crawlery, spidery, boti) jsou odfiltrovány ze statistik použití. Roboty detekujeme na základě standardizovaného seznamu robotů poskytovaného projekty COUNTER a Making Data Count .

Jak často aktualizujete statistiky použití?

Jednou denně.

Jak mohu zobrazit nejprohlíženější záznamy?

Každé vyhledávání na {{ config.THEME_SITENAME }} umožňuje seřadit výsledky vyhledávání podle " nejprohlíženějších ".

Jak sledujete?

Při sledování dodržujeme COUNTER Code of Practice stejně jako Code of Practice for Research Data Usage Metrics . To znamená, že naše statistiky použití lze porovnat s jinými repozitáři kompatibilními s COUNTER.

Jaký je rozdíl mezi strojem a robotem?

Strojový požadavek je automatizovaný požadavek iniciovaný lidským uživatelem, např. skript stahující data z {{ config.THEME_SITENAME }} a provádějící analýzu dat. Robotický požadavek je automatizovaný požadavek provedený např. crawlerem vyhledávače.

Jak anonymizujete uživatele?

Pro každou událost zobrazení/stažení sledujeme anonymizované ID návštěvníka. Toto anonymizované ID návštěvníka se pro uživatele mění každých 24 hodin, takže uživatel zobrazující stejný záznam ve dvou různých dnech bude mít dvě různá anonymizovaná ID návštěvníka. Důvodem, proč sledujeme anonymizované ID návštěvníka, je počítání unikátních zobrazení a stažení.

Z bezpečnostních důvodů také vedeme log přístupu k webovému serveru, který obsahuje vaši IP adresu a řetězec user agent vašeho prohlížeče. Tento log přístupu k webovému serveru je automaticky smazán po maximálně 1 roce a je také striktně oddělen od sběru statistik použití.

Anonymizované ID návštěvníka je generováno z osobního identifikátoru, jako je:

  1. ID uživatele,
  2. ID relace,
  3. nebo IP adresa a řetězec user-agent vašeho prohlížeče.

Kombinujeme osobní identifikátor s náhodnou textovou hodnotou (salt) a aplikujeme jednosměrnou kryptografickou hash funkci pro zakódování dat. Salt (náhodná textová hodnota) je zahozena a regenerována každých 24 hodin. Použití a následné zahození náhodného saltu zajišťuje, že anonymizované ID návštěvníka je plně náhodné.

Mohu se odhlásit ze sledování statistik použití?

Ne, není možné se odhlásit. Sledování statistik použití je plně anonymizované a probíhá na straně serveru.

Podporujete statistiky použití pro komunitu?

Zatím ne, ale budeme přidávat agregované statistiky použití pro vaše komunity.

Sledujete mé vyhledávací dotazy?

Ne.

Provádíte nějaké manuální nebo automatické profilování uživatelů?

Ne.
