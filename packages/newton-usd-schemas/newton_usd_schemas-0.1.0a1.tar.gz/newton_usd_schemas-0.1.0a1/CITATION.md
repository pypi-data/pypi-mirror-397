# Citation Guide

## To Cite Newton USD Schemas

If you use Newton USD Schemas in a publication, please use citations in the following format (BibTeX entry for LaTeX):

```tex
@software{newton-usd-schemas,
  title = {{Newton USD Schemas}: {USD} schema definitions for the {Newton} physics engine.},
  author = {{Newton Contributors}},
  year = {2025},
  url = {https://github.com/newton-physics/newton-usd-schemas},
  organization = {{Newton a Series of LF Projects, LLC}},
  license = {Apache-2.0}
}
```
