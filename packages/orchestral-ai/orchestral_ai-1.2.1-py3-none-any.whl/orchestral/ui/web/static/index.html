How to Use LaTeX Export

1

Create orchestral.tex in your Overleaf project

Download or copy the Orchestral LaTeX module to your project:

2

Add to your document's preamble

In your main .tex file, add this line after \documentclass:

\input{orchestral}
3

Copy panels using the TEX button

Click the TEX button on any message panel to copy its LaTeX code.

Paste it into your document and it will look like:

\begin{orchestralusermessage}
    Hello world!
\end{orchestralusermessage}

Agent messages, system prompts, and tool outputs each have their own styled environments!
