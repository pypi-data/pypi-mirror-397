# Outline Checklist Template

This is an example checklist for creating SEO article outlines.
Customize this file for your business needs.

## Structure Requirements

- [ ] Single H1 (title) that includes main keyword
- [ ] H2 sections for main topics (80% of content)
- [ ] H3 sub-sections for detailed breakdowns
- [ ] Supplemental content section (20% of content)

## Content Quality

- [ ] Address primary user search intent
- [ ] Include expertise-demonstrating sections
- [ ] Cover competitor topics + unique angles
- [ ] Logical flow from introduction to conclusion

## SEO Requirements

- [ ] Main keyword in H1
- [ ] Related keywords in H2s
- [ ] Natural keyword distribution
- [ ] FAQ section with question-based headings

## E-E-A-T Signals

- [ ] Experience: First-hand knowledge sections
- [ ] Expertise: Technical depth appropriate to topic
- [ ] Authoritativeness: Citations/references planned
- [ ] Trustworthiness: Accuracy verification points

## Formatting

- [ ] First heading connects to last heading
- [ ] Related headings grouped together
- [ ] Incremental lists where appropriate
- [ ] 80/20 main/supplemental content ratio
