# Article Checklist Template

This is an example checklist for writing SEO articles.
Customize this file for your business needs.

## Content Structure

- [ ] Compelling H1 title with main keyword
- [ ] Hook in first paragraph (problem/promise)
- [ ] Clear thesis statement
- [ ] Logical section progression
- [ ] Strong conclusion with call-to-action

## SEO Optimization

- [ ] Main keyword in first 100 words
- [ ] Keyword density 1-2%
- [ ] LSI keywords naturally included
- [ ] Internal linking opportunities marked
- [ ] External authority links planned

## Readability

- [ ] Short paragraphs (3-4 sentences max)
- [ ] Bullet points for lists
- [ ] Subheadings every 300 words
- [ ] Simple language (aim for grade 8 reading level)
- [ ] Active voice preferred

## E-E-A-T Compliance

- [ ] Author expertise demonstrated
- [ ] Sources cited where applicable
- [ ] Up-to-date information
- [ ] Balanced perspective presented
- [ ] Clear, accurate claims

## Engagement Elements

- [ ] Questions to reader
- [ ] Examples and case studies
- [ ] Data/statistics included
- [ ] Actionable takeaways
- [ ] Visual content suggestions

## Technical

- [ ] Meta description drafted
- [ ] Alt text for images planned
- [ ] Word count meets target
- [ ] No duplicate content
- [ ] Grammar and spelling checked c
